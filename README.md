# fordix
# This is the node.js backend for fordix software    

## To install packages
- "npm install" or "npm i"

## To start server
- "npm app.js"
- Or first install nodemon globaly it will restart servrs whenever file gets save
- "npm i nodemon  -g"
- "nodemon app.js"

## server will start on defined port (default - 3000)
